# 🎵 Spotify Mood Prediction

This project is a machine learning web application that predicts the mood of a song using Spotify audio features.

## 🚀 Features
- Predicts mood using Logistic Regression
- Uses Spotify audio features like energy, valence, tempo
- Interactive web app built with Streamlit
- Visualizations: scatter plot, confusion matrix, ROC curve

## 📊 Dataset
- Spotify audio features dataset
- Features used:
  - danceability
  - energy
  - loudness
  - speechiness
  - acousticness
  - instrumentalness
  - liveness
  - valence
  - tempo

## 🧠 Model
- Logistic Regression
- Accuracy evaluated using:
  - Accuracy score
  - Confusion Matrix
  - ROC Curve

## 💻 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
